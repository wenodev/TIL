# 데이터베이스
## 데이터베이스 종류
1. RDS : 일반적인 RDS 사용할때 사용함
2. Aurora : 엔터프라이즈급에서 RDS 사용할때 사용함
3. DynamoDB : NoSQL 사용할때 사용함
4. Redshift : 데이터웨어하우스 관리

## EC2에 디비 운영 VS AWS 데이터베이스 서비스 사용
- 성능에 초점
- 관리 및 설정이 더 쉬움
