# 6-3 변수의 유효범위와 클로저

- 자바스크립트는 함수 지향 언어이다.

## 코드 블록 
- 중괄호 "{}" 단위로 코드 블록이 나누어 진다.
- 일반 함수, for, while, if 등등

## 중첩 함수
- 함수 내부에서 선언한 함수를 말한다.
```javascript
function makeCounter() {
    let count = 0;
    return function () {
        return count++;
    }
}

let counter = makeCounter();

alert( counter() ); // 0
alert( counter() ); // 1
alert( counter() ); // 2
```
