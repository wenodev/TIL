# 2-15 함수
## return문이 없거나 return만 있는 함수는 'undefined'를 반환한다.
```js
function doNothing(){
    
}
alert(doNothing()) // undefinded
```
```js
function doNothing(){
    return;
}
alert(doNothing()) // undefinded
```
## return 다음 개행문자를 넣으면 안된다
- 자바스크립트는 return 뒤에 자동으로 세미콜론이 만들어져 return 이후의 코드는 실행되지 않습니다.
