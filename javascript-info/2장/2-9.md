# 2-9 비교 연산자
## 문자열 비교
- 사전편집(lexicographical)순이다. -> 정확히는 유니코드 순이다.
- 사전 뒤쪽의 문자열이 사전 앞쪽의 문자열보다 크다고 판단한다.
- 같은 문자열의 경우 길이가 더 긴 문자열이 크다고 판단한다.
- 대소문자의 경우 소문자가 크다고 판단한다.
```javascript
"A" < "Z" // true
"AB" > "AC" // false
"AB" < "ABC" // true
```

## 다른 형르 가진 값 간의 비교
String VS Number
```javascript
'2' > 1 // true
'01' == 1 // true
```

## 일치 연산자, 동등 연산자
- 일치 연산자(===)는 자료형의 동등 여부까지 판단.
- 동등 연산자(==)는 피연산자를 숫자형으로 바꾼다.
