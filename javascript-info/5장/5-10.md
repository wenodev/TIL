# 구조 분해 할당

## 배열 분해하기
- 구조 분해 할당
```javascript
//배열 선언
let arr = ["Weno", "Son"]

// 구조 분해 할당
let [firstName, surName] = arr;

alert(firstName); // Weno
alert(surName); // Son
```
- split으로 분해
```javascript
let [firstName, surname] = "Weno Son".split(` `);
```

## '...'로 나머지 요소 가져오기
- 배열의 나머지 부분을 한번에 저장할 떄 사용한다.
```javascript
let [name1, name2, ...rest] = ["Julius", "Caesar", "Consul", "of the Roma Republic"];

alert(name1); // "Julius"
alert(rest[0]); // "Consul
alert(rest.lengh); // 2
```
