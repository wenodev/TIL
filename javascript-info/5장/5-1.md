# 5-1 원시값의 메서드
- 자바스크립트는 원시값에서도 메서드를 호출할 수 있다.
- 원시값
    - string, number, bigint, boolean, symbol, null, undefined
    
- 객체
    - 프로퍼티가 존재하고 프로퍼티는 키와 값으로 구성된다.
    - 함수를 프로퍼티로 저장할 수 있따.
    ```javascript
  {
        name: "weno"
        writeCode = function() {
        }         
   }
    ```
